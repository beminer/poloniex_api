--- Важно ---
В скрипте идет работа с биржей Poloniex, но саму функцию "api_query" можно использовать и для других бирж. Скорее всего нужно будет изменить только ссылку для запроса.
_____________________

Файл "config.py" содержит две переменные, которые нужно скопировать с биржи Private-key и Secret (на разных биржах название может быть разным) и вставить в файл.

Добавлена функция telegram_update к видео от 13.02.2018

В файле config не забудьте указать токен, который дается за создание бота в телеграме у @botfather (см. видео https://goo.gl/rCSRD5)
